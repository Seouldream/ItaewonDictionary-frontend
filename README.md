## 💻 기능
- 고객
  - 말하고 첨삭받기 게시물 등록
  - 실전 템플릿 영작 후 예시 답안 비교
  - 회원들간 커뮤니티 활동 및 스터디 모집
  - 1분 템플릿 회화 연습 및 강의 시청 
- 어드민
  - 게시물 삭제
  - 게시물 수정
  - 카테고리 관리
